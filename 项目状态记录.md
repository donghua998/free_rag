# 知识库问答项目状态记录

## 项目概述
基于 RAG（检索增强生成）技术的知识库问答系统，使用 Spring Boot + MyBatis-Plus + LangChain4j 构建。

---

## 技术栈

| 技术 | 版本 | 用途 |
|------|------|------|
| Spring Boot | 3.3.2 | 基础框架 |
| MyBatis-Plus | 3.5.9 | ORM框架（spring-boot3-starter）|
| LangChain4j | 0.36.2 | AI能力集成（核心库，无starter）|
| JWT | 0.12.3 | 认证令牌 |
| MySQL | 8.0 | 业务数据库（端口3307）|
| Milvus | 2.4.4 | 向量数据库（端口29530）|
| MinIO | - | 对象存储（Milvus依赖，端口19000）|
| etcd | 3.5.5 | 服务发现（Milvus依赖，端口22379）|
| Lombok | 1.18.30 | 简化代码 |

## 注意事项

1. **端口冲突**：MySQL使用3307，Milvus使用29530，避免与本地其他服务冲突
2. **Docker服务**：启动顺序 etcd → MinIO → Milvus
3. **权限注解**：使用 `@RequireRole` 或 `@RequirePermission` 标记需要权限的接口
4. **Token传递**：在Header中使用 `Authorization: Bearer <token>`
