## 模块职责说明

| 模块 | 职责描述 |
|------|----------|
| **config** | 全局配置类，包括安全、数据库、缓存、JWT、异步任务等 |
| **common** | 公共组件：常量、工具类、异常处理、统一返回结果、基础实体 |
| **user** | 用户注册、登录、个人信息管理 |
| **auth** | 角色权限管理、JWT认证、权限校验注解和切面 |
| **kb** | 知识库CRUD、文档上传、文本分块处理 |
| **qa** | 问答会话管理、消息历史、流式响应、AI API客户端 |
| **retrieval** | 向量嵌入生成、向量数据库存储、相似度检索 |
| **api** | 对外API接口、API Key管理、调用配额限制、限流 |

---

## 核心功能流程

```
用户注册/登录 (user + auth)
    ↓
创建知识库 (kb)
    ↓
上传文档 → 文本分块 → 向量化存储 (kb + retrieval)
    ↓
发起问答 → 检索相关知识 → 调用AI API (qa + retrieval)
    ↓
返回答案 (流式/非流式)
```

## 技术栈

- **框架**: Spring Boot 3.x
- **ORM**: MyBatis-Plus
- **数据库**: MySQL 8.0
- **缓存**: Redis
- **向量数据库**: Milvus / PGVector
- **安全**: Spring Security + JWT
- **AI API**: OpenAI / 通义千问 / Claude
