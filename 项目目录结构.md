# AI知识库系统 - 项目目录结构

> 技术栈：Spring Boot 4.1.0 + Java 17 + LangChain4j

---

## 📁 目录结构

```
src/main/java/com/example/quanxiankongzhi/
├── api/                          # 🔌 API管理 - 第三方AI服务配置
├── auth/                         # 🔐 认证授权 - 登录、JWT、权限
├── common/                       # 📦 公共模块
│   └── result/
│       └── Result.java           # 统一响应结果
├── config/                       # ⚙️ 全局配置
├── kb/                           # 📚 知识库 - Knowledge Base管理
├── qa/                           # 💬 问答系统 - Q&A对话
├── retrieval/                    # 🔍 检索模块 - 向量检索、Embedding
├── user/                         # 👤 用户管理
└── QuanxiankongzhiApplication.java
```

---

## 📂 各模块详细结构

### api - API管理
```
api/
├── controller/ApiController.java
├── service/ApiService.java
├── service/impl/ApiServiceImpl.java
├── entity/ApiConfig.java
├── dto/ApiAddRequest.java
├── dto/ApiTestRequest.java

### auth - 认证授权
```
auth/

### common - 公共模块

common/

## 🗄️ 数据库表

```sql
-- 用户表
sys_user (id, username, password, email, avatar, status, create_time, update_time)

-- API配置表  
api_config (id, api_name, api_url, api_type, api_key, model_name, temperature, max_tokens, is_default, status)

-- 知识库表
knowledge_base (id, kb_name, description, user_id, doc_count, status)

-- 文档表
kb_document (id, kb_id, doc_name, doc_type, doc_path, chunk_count, status)

-- 对话历史表
chat_history (id, session_id, user_id, kb_id, question, answer, create_time)
```
